# Personal Website

Source code of my personal website — a bilingual (EN / 中文) single-page portfolio covering research, education, experience, skills and contact info.

Work in progress. 🚧

## Tech Stack

- [Astro](https://astro.build) 6
- [Tailwind CSS](https://tailwindcss.com) v4
- Deployed via Cloudflare Pages & GitHub Pages

## Local Development

```bash
npm install
npm run dev        # → http://localhost:4321
npm run build      # → ./dist
```

All site copy lives in [`src/data/content.ts`](src/data/content.ts).
